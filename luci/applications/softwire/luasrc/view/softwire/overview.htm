Softwire configuration

This section of the Web GUI is dedicated to tunnel configuration: Softwire (IPv6 over IPv4) or DS-Lite (IPv4 over IPv6). Related topics - fragmentation, MTU, services and tunnel status, portproxy (a tool used for automatic port configuration, using NAT-PMP and UPnP) and manual port forwarding configuration are also available.
